# Vue 브라우저 기반 트랜스코딩

- 사용 라이브러리: https://github.com/ffmpegwasm/ffmpeg.wasm 
- API: https://github.com/ffmpegwasm/ffmpeg.wasm/blob/master/docs/api.md

## 실행하기
```
npm install
set export NODE_OPTIONS=--openssl-legacy-provider
npm run serve
```

### 빌드
```
npm run build
```

### 린팅
```
npm run lint
```
